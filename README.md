# homeexam2k24

This is the home exam for the course [D7032E Software engineering](https://www.ltu.se/utbildning/kurs/d70/d7032e-programvaruteknik) for fourth year students in Computer Science at Lule√• tekniska universitet.

## How to run

1. Download the repo files

2. While inside the repo folder, run `chmod +x compile.sh server.sh client.sh` in your terminal

3. Run `./compile.sh` 

4. Run `./server.sh` to start and host the server

5. Run `./client.sh` to connect to the server from another client
