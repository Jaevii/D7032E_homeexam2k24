# homeexam2k24

This is the home exam for the course [D7032E Software engineering](https://www.ltu.se/utbildning/kurs/d70/d7032e-programvaruteknik) for fourth year students in Computer Science at Lule√• tekniska universitet.

## How to run

1. Download the repo files

2. While inside the repo folder, run in your terminal:
```bash
chmod +x compile.sh server.sh client.sh
```

3. Compile with:
```bash
./compile.sh
``` 

4. To start and host the server, run:
```bash
./server.sh
```

5. To connect to the server from another client, run:
```bash
./client.sh
```

## Testing

Run tests with:
```bash
./runTests.sh
```
